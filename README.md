# Credit Risk Analytics & GenAI Assistant

## Descripción

Este proyecto simula una solución de Advanced Analytics para una entidad financiera.

El objetivo es desarrollar un modelo de riesgo crediticio interpretable y complementarlo con un asistente basado en IA Generativa para apoyar decisiones de negocio.

---

## Objetivos

- Analizar datos de clientes
- Identificar factores asociados al riesgo crediticio
- Entrenar modelos de Machine Learning
- Explicar las predicciones mediante SHAP
- Construir un dashboard interactivo
- Generar recomendaciones utilizando IA Generativa

---
